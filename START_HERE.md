## 🚀 Quick Start (5 Minutes)

### Step 1: Setup
```bash
# Double-click this file:
SETUP_EVERYTHING.bat
```

### Step 2: Start Server
```bash
cd server
node index.js
```

### Step 3: Start Admin Panel
```bash
cd admin-panel
npm start
```

**Done!** Desktop window opens automatically. 🎉
