College Attendance Admin Panel
Standalone Installer Package
========================================

VERSION: 1.0.0
SIZE: ~71 MB
PLATFORM: Windows 7/8/10/11 (64-bit)

========================================
QUICK START
========================================

1. Double-click: "College Attendance Admin Setup 1.0.0.exe"

2. If Windows shows "Windows protected your PC":
   - Click "More info"
   - Click "Run anyway"
   (This is normal - the app is not code-signed)

3. Follow the installation wizard:
   - Accept license
   - Choose installation folder
   - Select shortcuts
   - Click Install

4. Launch from Desktop shortcut or Start Menu

========================================
FIRST TIME SETUP
========================================

After installation:

1. Open "College Attendance Admin"

2. Click Settings (⚙️ icon in top right)

3. Enter your server URL:
   Examples:
   - Local PC: http://localhost:3000
   - Same Network: http://192.168.x.x:3000
   - Remote Server: http://your-server.com:3000

4. Click Save

5. Start managing students, teachers, and timetables!

========================================
TROUBLESHOOTING
========================================

Problem: "Windows protected your PC"
Solution: Click "More info" → "Run anyway"

Problem: Cannot connect to server
Solution: 
  - Verify server is running
  - Check server URL in Settings
  - Check firewall settings

Problem: App won't start
Solution:
  - Right-click installer → Run as Administrator
  - Reinstall the application

========================================
UNINSTALL
========================================

To remove the admin panel:

Method 1: Windows Settings
  Settings → Apps → College Attendance Admin → Uninstall

Method 2: Control Panel
  Control Panel → Programs → Uninstall a program

Method 3: Start Menu
  Start Menu → College Attendance → Uninstall
