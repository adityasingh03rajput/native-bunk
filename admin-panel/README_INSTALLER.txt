========================================
College Attendance Admin Panel
Windows Installer
========================================

VERSION: 1.0.0
PLATFORM: Windows 7/8/10/11 (64-bit)

========================================
INSTALLATION INSTRUCTIONS
========================================

1. Double-click "College Attendance Admin Setup 1.0.0.exe"

2. Follow the installation wizard:
   - Accept the license agreement
   - Choose installation directory
   - Select shortcuts (Desktop, Start Menu)
   - Click Install

3. Launch the application:
   - From Desktop shortcut, OR
   - From Start Menu > College Attendance

========================================
FIRST TIME SETUP
========================================

1. Open the Admin Panel

2. Click Settings (⚙️ icon)

3. Enter your server URL:
   - Local: http://localhost:3000
   - Network: http://192.168.x.x:3000
   - Remote: http://your-server.com:3000

4. Click Save

5. Start managing your college attendance system!

========================================
SYSTEM REQUIREMENTS
========================================

- Operating System: Windows 7/8/10/11 (64-bit)
- RAM: 2GB minimum, 4GB recommended
- Disk Space: 500MB free space
- Network: Internet/LAN connection required
- Backend Server: Must be running

========================================
TROUBLESHOOTING
========================================

Problem: "Windows protected your PC" warning
Solution: Click "More info" → "Run anyway"

Problem: Cannot connect to server
Solution: 
  - Verify server is running
  - Check server URL in Settings
  - Check firewall settings

Problem: App won't start
Solution:
  - Run as Administrator
  - Reinstall the application

========================================
UNINSTALLATION
========================================

Method 1: Windows Settings
  - Settings > Apps > College Attendance Admin > Uninstall

Method 2: Control Panel
  - Control Panel > Programs > Uninstall a program
  - Select "College Attendance Admin" > Uninstall

Method 3: Start Menu
  - Start Menu > College Attendance > Uninstall
