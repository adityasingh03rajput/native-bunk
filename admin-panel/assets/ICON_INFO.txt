Icon Files Needed for Building Executables
==========================================

Place your icon files here:

1. icon.png (512x512 or 1024x1024) - For Linux AppImage
2. icon.ico (256x256 with multiple sizes) - For Windows .exe
3. icon.icns (512x512@2x) - For macOS .dmg

You can create these from a single PNG using online tools:
- https://www.icoconverter.com/ (for .ico)
- https://iconverticons.com/ (for .icns)
- Or use any graphic design software
